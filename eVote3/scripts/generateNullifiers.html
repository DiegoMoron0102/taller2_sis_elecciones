📋 Array para registerNullifiers()

📖 Instrucciones de Uso

  1. Copia el array de nullifiers (botón "Copiar Array")
  2. En Remix, despliega el contrato con los candidatos deseados
  3. Llama a registerNullifiers() y pega el array
  4. Llama a openElection() para abrir la votación
  5. Usa los nullifiers generados para llamar a castVote(nullifier, candidateIndex)
  6. Verifica que no se puede votar dos veces con el mismo nullifier
  7. Lee los resultados con getCandidate(index)
